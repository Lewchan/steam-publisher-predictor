## 本轮变更摘要

### P2 — 校准种子游戏对比增强（Streamlit 版）

对标 index.html 校准面板的可视化深度，增强 app.py 中的校准种子游戏对比模块：

#### 1. 四维偏差可视化 ✅
- 新增**四维偏差对比图**（销量/质量/CL/用户池四轴），用 Streamlit bar_chart 呈现
- 蓝色柱 = 负偏差（模型低估），橙色柱 = 正偏差（模型高估），以 0% 为理想值
- 支持 `-50%` 到 `+50%` 的完整偏差范围

#### 2. 逐维度偏差进度条 ✅
- 对每个种子游戏的四个指标分别渲染进度条
- 颜色编码：🟢 < 20% | 🟡 20-50% | 🔴 > 50%
- 双栏布局展示所有游戏

#### 3. 逐游戏结构化详情卡片 ✅
- 替代原来 raw JSON 展示
- 顶部三列 Metric 卡片：销量预测 / 质量评分 / CL 分数
- 用户池规模单列展示
- 四维偏差卡片：每维度独立展示，颜色编码 + 百分比
- 可选展开查看原始 JSON 数据

#### 4. 校准结果导出 ✅
- 新增导出按钮，生成带时间戳的 CSV 文件
- 包含所有游戏的全维度偏差数据
